Tutorial 5 - Deities

Introduction

In the last tutorial, we added a new class that could cast spells. This time we're going back to something simpler - adding new deities for your game world.

This tutorial assumes you're familiar with the contents of the first and second tutorials, in which you added a feat and a race.

Loading the Editor

As usual, load the editor from the Hero Lab "Tools" menu and re-open the "tutorial" file that we've been working on so far.

Create the deity

Find the "Deity" tab under the "General" category. We're going to add one of the gods of our home campaign, Mystika - the God of Magic - with similar information as found in the Player's Handbook for deities.

Create a new, blank deity named Mystika, and give it an appropriate unique id. While there are many facets of deity information you can fill in, we're interested in Mystika's Alignment, domains, and symbol. Selecting the alignment is easy - just pick it from a menu.

Mystika is the god of magic, so his suggested domain is the Knowledge domain. Click the "Edit" button beside "Suggested Domains" and select it from the list. This adds the domain tag to the hero - if you don't see the domain you want here, choose "- New Tag -" instead, and you can create a new domain.

Next, type Mystika's portfolio, the "God of magic", into the "Portfolio" text box. If you want to enter more text than will fit, you can click the "Edit" button to get a bigger window to type into. You can also enter the appropriate symbol text in the "Minor Details" section below, along with anything else you want to specify.

We're half-way done - Mystika now appears at the bottom of our list of deities.

Let's add a couple of new things - a setting restriction for Mystika, so he only appears for games in our "Forgotten Worlds" setting, and make him appear at the top of the list.

To add a setting restriction, we want to set an appropriate Source on Mystika. Scroll to the bottom of the list of facets of the deity, and you'll see a button for Sources. Click "Edit" and you'll be able to see a list of all the sources defined for the game so far. Initially, this will just be the list of built-in sources.

We need to create a new one for our game, so check the - New Source - option at the top, and hit "OK". Now we get to enter the information about our game - enter a name for the source, which will appear in Hero Lab, and a unique id. (The unique id should be a shortened form of the name, and should include your user identifier, just like the unique id of the things you've been creating.)

Leave the parent source specified as "UserParent" - this will display it at the top of the list of sources on the Configure Hero form in Hero Lab.

Hit OK, and you've created a new source! Now you've created it here, you'll be able to re-use it on other deities and content you create for your world - it will be selectable in the list of sources for any new thing you create.

Test the updated thing, and you'll notice that Mystika no longer appears in the list of deities on the Background tab. Don't worry, that's expected! We haven't told Hero Lab to allow content from our Forgotten Worlds setting yet - let's fix that.

Go to the Hero menu within Hero Lab, and choose Configure Hero. You'll see a new option in the list of settings on the right - our new Forgotten Worlds setting.

Turn the Forgotten Worlds setting on, and Mystika returns to list of deities! Now we just have to move him to a more appropriate place. Go back to the editor, and find the "Deity Category Order Group" setting - change that to an early number, like 5. (This sets a tag that Hero Lab uses to decide which order the list of deities should appear in.)

Now test the thing again, and you'll see Mystika appearing proudly at the top of the list of deities! If you want to arrange different groups of deities separately from each other, you can set their "Deity Category Order Group" values differently - the lower the number, the earlier the deity will appear in the list.
